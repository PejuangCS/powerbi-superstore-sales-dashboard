## 🎯 Project Overview
Dashboard Business Intelligence interaktif yang dibuat menggunakan Power BI untuk memonitor performa penjualan (*Total Sales* & *Total Orders*), menganalisis pencapaian target per wilayah menggunakan formula DAX `DIVIDE`, serta melihat persebaran pesanan di AS menggunakan model *Star Schema*.
